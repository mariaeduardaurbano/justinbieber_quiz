## 📱 Sobre o Projeto

O **Justin Bieber Quiz** é um aplicativo mobile desenvolvido em **Flutter** que testa o quanto você sabe sobre o cantor canadense Justin Bieber. Desde sua infância até os maiores hits da carreira, o quiz abrange curiosidades, discografia, vida pessoal e muito mais!

---

## ✨ Funcionalidades

- 🎯 Perguntas de múltipla escolha sobre Justin Bieber
- ⏱️ Sistema de pontuação ao final do quiz
- 🎨 Interface moderna e temática
- 📊 Resultado de 0 a 10
- 🔄 Opção de reiniciar e jogar novamente

---

## 🚀 Como Rodar o Projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado na sua máquina:

- [Flutter SDK](https://flutter.dev/docs/get-started/install) (versão 3.x ou superior)
- [Dart SDK](https://dart.dev/get-dart)
- [Android Studio](https://developer.android.com/studio) ou [VS Code](https://code.visualstudio.com/)
- Um emulador Android/iOS ou dispositivo físico

### Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/justin-bieber-quiz.git

# Acesse a pasta do projeto
cd justin-bieber-quiz

# Instale as dependências
flutter pub get

# Execute o aplicativo
flutter run
```
